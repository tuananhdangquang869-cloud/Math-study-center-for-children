CHƯƠNG TRÌNH HỌC

Lộ trình từ Lớp 1 đến Lớp 5

Bám sát chương trình Bộ GD&ĐT — mỗi khối lớp có mục tiêu rõ ràng và bài tập đa dạng.

🌱

Lớp 1

Xây dựng nền tảng số học vững chắc, giúp trẻ tự tin làm toán cơ bản.

Cộng trong phạm vi 10

  • ✓ Nhận biết số 1–10
  • ✓ Phép cộng đơn giản
  • ✓ Bài tập điền số

Trừ trong phạm vi 10

  • ✓ Phép trừ cơ bản
  • ✓ So sánh số
  • ✓ Bài tập thực tế

Cộng, trừ đến 20

  • ✓ Cộng có nhớ
  • ✓ Trừ có mượn
  • ✓ Bài toán có lời
🌿

Lớp 2

Mở rộng phạm vi số đến 100 và làm quen với phép nhân.

Cộng, trừ trong 100

  • ✓ Cộng nhiều chữ số
  • ✓ Trừ có nhớ
  • ✓ Ước tính kết quả

Bảng nhân 2, 3, 4, 5

  • ✓ Ý nghĩa phép nhân
  • ✓ Bảng nhân thuộc lòng
  • ✓ Vận dụng thực tế

Đo lường cơ bản

  • ✓ Đọc giờ đồng hồ
  • ✓ Đo độ dài
  • ✓ Bài toán tiền
🌳

Lớp 3

Thành thạo 4 phép tính cơ bản và tiếp cận hình học phẳng.

Bảng nhân 6, 7, 8, 9

  • ✓ Bảng nhân nâng cao
  • ✓ Nhân với 0 và 1
  • ✓ Tính nhẩm

Phép chia

  • ✓ Chia hết
  • ✓ Chia có dư
  • ✓ Quan hệ nhân-chia

Hình học

  • ✓ Chu vi hình chữ nhật
  • ✓ Chu vi hình vuông
  • ✓ Điểm, đoạn thẳng
⭐

Lớp 4

Làm việc với số lớn, phân số và hình học không gian cơ bản.

Số có nhiều chữ số

  • ✓ Đọc, viết số lớn
  • ✓ Nhân chia số lớn
  • ✓ Bài toán nâng cao

Phân số

  • ✓ Khái niệm phân số
  • ✓ So sánh phân số
  • ✓ Cộng trừ cùng mẫu

Góc và đường thẳng

  • ✓ Góc nhọn, tù, vuông
  • ✓ Hai đường song song
  • ✓ Đo góc
🚀

Lớp 5

Hoàn thiện kiến thức tiểu học, chuẩn bị nền tảng cho THCS.

Số thập phân

  • ✓ Đọc viết thập phân
  • ✓ Cộng trừ thập phân
  • ✓ Nhân chia thập phân

Phần trăm

  • ✓ Khái niệm %
  • ✓ Tìm % của một số
  • ✓ Bài toán % thực tế

Diện tích & thể tích

  • ✓ Diện tích hình chữ nhật
  • ✓ Diện tích hình tròn
  • ✓ Thể tích hình hộp
